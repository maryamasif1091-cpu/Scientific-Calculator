# 🧮 Scientific Calculator (Vite + TypeScript)

A modern, fully functional scientific calculator built using Vite and TypeScript. It supports advanced mathematical operations with a clean dark UI and real-time expression evaluation similar to a CASIO-style calculator.

---

##  Features

- Basic arithmetic operations (+, −, ×, ÷)
- Scientific functions: sin, cos, tan (degree mode)
- Square root (√), logarithm (log), natural log (ln)
- Power operator (^)
- Constant support (π)
- Backspace (delete last input)
- Clear all functionality
- Real-time expression evaluation
- Clean dark professional UI
- Fast performance using Vite

##  Installation & Setup

1. Clone repository:
git clone https://github.com/maryamasif1091-cpu/Scientific-Calculator.git

2. Go to project folder:
cd Scientific-Calculator

3. Install dependencies:
npm install

4. Run development server:
npm run dev

5. Open in browser:
http://localhost:5173
